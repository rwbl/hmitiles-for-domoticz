# Blueprint HMITiles Workbench

This directory serves as the isolated development and testing environment for new HMI tiles.

## Screenshots

![HMITiles Workbench](hmitiles-workbench.png)

---

### Workflow:
1. **Define:** Add or modify HMITile structures inside `hmitiles.index`.
2. **Verify:** Open `index.html` within Domoticz to preview, test responsiveness, and validate the decoupled HTML5/CSS3/JS rendering before pushing to production custom pages.
